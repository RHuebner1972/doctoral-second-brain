# Doctoral Second Brain

A Claude Code plugin providing 31 skills for doctoral learners â€” from first-year coursework through dissertation defense.

## Installation

### From this repo (self-hosted marketplace)

```
/plugin marketplace add <your-github-username>/doctoral-second-brain
/plugin install doctoral-brain@doctoral-brain
```

### Local install (for testing)

```bash
claude --plugin-dir /path/to/doctoral-second-brain
```

## Skills

Once installed, invoke any skill with `/doctoral-brain:<skill-name>`.

### Research & Scholarship
| Skill | What it does |
|---|---|
| `research-gap` | Find white space in a field â€” population, methodological, contextual, and temporal gaps |
| `problem-statement` | Craft a tight, evidence-grounded problem statement |
| `research-questions` | Generate and refine central and sub-questions aligned with methodology |
| `literature-review` | Plan, structure, and draft a literature review |
| `source-synthesis` | Extract themes, tensions, and convergences across multiple sources |
| `methodology` | Select and justify a research design |
| `theoretical-framework` | Identify and apply a theoretical or conceptual framework |
| `dissertation-outline` | Scaffold a full five-chapter dissertation |
| `chapter-draft` | Generate scholarly first drafts of any chapter or section |
| `apa-citations` | Format citations and references in APA 7th edition |
| `peer-review-response` | Draft point-by-point responses to committee feedback |
