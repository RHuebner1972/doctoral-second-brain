# Source Synthesis

## Purpose
Transform a collection of sources from a list of individual findings into a synthesized argument — extracting convergence, divergence, and gaps across the set.

## What Synthesis Means
Synthesis is the intellectual act of identifying what multiple sources collectively say about a topic — where they agree, where they conflict, what they collectively miss, and how they build on each other.

It is the difference between:
- **Summary:** "Source A says X. Source B says Y. Source C says Z."
- **Synthesis:** "Sources A, B, and C collectively argue X, though B and C challenge A's assumption about Y, suggesting that Z may be context-dependent."

## Process

### Step 1 — Extract From Each Source
For each source provided, extract:
- **Core argument or finding** (one sentence)
- **Methodology** (how they studied it)
- **Population/context** (who/where)
- **Key constructs or variables**
- **Limitations noted by the authors**

### Step 2 — Build the Synthesis Matrix
Create a comparison grid:

| Source | Core Finding | Method | Population | Key Construct | Limitation |
|--------|-------------|--------|------------|---------------|-----------|
| Author (year) | | | | | |

### Step 3 — Identify Patterns

**Convergence:** Where do 3+ sources agree?
> "A consistent finding across the literature is..."

**Divergence/Tension:** Where do sources contradict or qualify each other?
> "While [Author A] found X, [Author B] reported Y, a difference potentially attributable to..."

**Evolution:** How has thinking on this topic changed over time?
> "Early work focused on [A] (Author, year), while more recent scholarship has shifted toward [B]..."

**Methodological pattern:** What methods dominate, and what does that shape/limit?
> "The preponderance of survey-based studies limits insight into the lived experience of..."

**Gaps the sources collectively leave:** What question cannot be answered by these sources combined?

### Step 4 — Write the Synthesis Narrative
Structure:
1. **Opening claim** — what this body of literature collectively establishes
2. **Convergent findings** — what is agreed upon (multi-citation)
3. **Tensions and debates** — where sources disagree and why
4. **Methodological patterns** — how the research has been conducted and what that shapes
5. **Collective gap** — what this set of sources cannot answer

### Step 5 — Link to the Study
Draw the explicit line:
> "Taken together, these sources establish [foundation], yet leave [gap] unaddressed. The present study addresses this gap by..."

## Output Format
1. **Per-source extraction table**
2. **Synthesis matrix** (comparative grid)
3. **Pattern analysis** (convergence, divergence, evolution, methodological, gaps — each named and evidenced)
4. **Draft synthesis paragraph(s)** (ready for dissertation use)
5. **Recommended additional sources** (if gaps in coverage are apparent)
