# Idea Capture

## Purpose
Instantly format a raw idea — however rough — into a structured, retrievable artifact that connects to the dissertation and doesn't get lost.

## The Problem This Solves
Doctoral learners have ideas constantly: in the shower, during a commute, mid-meeting, half-asleep. These ideas are often the most generative moments of the entire doctoral journey — unexpected connections, new framings, methodological insights.

But unrecorded ideas evaporate. Recorded-but-unstructured ideas pile up and become noise.

This skill turns a raw idea into a formatted knowledge artifact in under two minutes.

## The Idea Capture Format

```
DATE: [Today]
IDEA: [The raw idea in one sentence — as fast as possible, no polish]
TYPE: [Research / Methodology / Writing / Framework / Question / Connection / Other]
CONNECTED TO: [Dissertation section, RQ, theme, or source this relates to]
POTENTIAL USE: [Where or how this could show up in the dissertation]
STATUS: [Raw / Needs development / Ready to act on]
NEXT ACTION: [If this idea has legs — what's the next step?]
```

## Idea Types

**Research idea:** A new angle on the gap, a new population to consider, a new construct to explore
> "What if I expanded the study to include adjunct faculty, not just tenure-track?"

**Methodology idea:** A new way to collect or analyze data
> "What if I used a focus group as a member-checking strategy instead of individual follow-up interviews?"

**Writing idea:** A way to frame an argument, a metaphor, a better opening line
> "The introduction could open with the statistic about faculty turnover rates — more compelling than the current abstract opener"

**Framework idea:** A connection between two theories or constructs not previously linked
> "Social exchange theory and servant leadership both assume reciprocity — this could be the connecting argument in the conceptual framework"

**Question idea:** Something the learner doesn't know yet but wants to investigate
> "Does the literature on servant leadership in healthcare settings map onto higher education? It should — check."

**Connection idea:** A link between a source already read and an emerging finding or argument
> "That thing Smith (2020) said about psychological safety is exactly what Participant 3 described — this is a key evidence-to-literature connection for Chapter 5"

## Process

### Step 1 — Capture Fast
The priority is speed, not polish. Get the idea out in one sentence before it fades.

If the learner provides a rough idea:
- Accept it exactly as stated
- Don't ask clarifying questions yet — capture first, refine after

### Step 2 — Format It
Apply the idea capture template. Fill in what's known; leave unknowns blank.

### Step 3 — Develop (Optional, If Time Permits)
If the learner wants to develop the idea further:
- Ask: "What would it look like if this idea were implemented?"
- Ask: "What would need to be true for this to work?"
- Ask: "What's the next smallest step?"

### Step 4 — Connect to the Dissertation
For every idea, ask:
- Which chapter or section does this touch?
- Which research question does this relate to?
- Is this a new direction (possible scope change — flag it) or a refinement?

### Step 5 — File It
Recommend where to file based on type:
- Research idea → Knowledge gap journal if unexplored; note to chair if it changes scope
- Methodology idea → Chapter 3 parking lot
- Writing idea → Chapter parking lot for the relevant chapter
- Framework idea → Theoretical framework notes
- Connection idea → Chapter 5 discussion notes; mark for use in analysis
- Question idea → Knowledge gap journal

## The Capture-Don't-Chase Rule
When capturing an idea mid-work:
- **Do:** Write it in the parking lot or idea capture log
- **Don't:** Chase it immediately — it will break the current session's focus

The capture is the action. The chase happens in the next session.

## Output Format
1. **Formatted idea entry** (all fields filled)
2. **Idea type** (classified)
3. **Dissertation connection** (where this lives or could live)
4. **Next action** (if the idea has legs — one sentence)
5. **Filing recommendation** (where to put it in the second brain)
