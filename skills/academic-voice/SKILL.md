# Academic Voice

## Purpose
Edit or rewrite content to meet doctoral-level scholarly register — precise, objective, hedged where appropriate, and free of informal language.

## What Academic Voice Is

Academic writing is:
- **Precise** — uses exactly the right word, not an approximate one
- **Objective** — grounds claims in evidence, not opinion
- **Hedged appropriately** — acknowledges uncertainty where it exists ("may," "suggests," "appears")
- **Formal** — avoids contractions, slang, colloquialisms, and casual phrasing
- **Impersonal in most sections** — minimizes first person except where required (positionality, reflexivity)
- **Clear** — complexity of thought, not complexity of syntax

## Common Problems and Fixes

### Contractions
| Informal | Academic |
|----------|---------|
| don't | do not |
| can't | cannot |
| it's | it is |
| wouldn't | would not |

### Colloquialisms and Vague Language
| Informal | Academic |
|----------|---------|
| a lot of | a substantial number of / considerable |
| get | obtain / acquire / achieve |
| show | demonstrate / indicate / suggest |
| big | substantial / considerable / significant |
| really important | critically important / particularly salient |
| things | factors / elements / constructs / variables |
| looked at | examined / investigated / analyzed |
| talks about | discusses / addresses / examines |
| use | utilize (only when meaning "put to use for a specific purpose") |

### Hedging (Qualifying Uncertainty)
Use hedged language when reporting others' findings or making inferences:

| Too Strong | Hedged |
|------------|--------|
| "This proves..." | "This suggests..." / "This indicates..." |
| "X causes Y" | "X may contribute to Y" / "X is associated with Y" |
| "All leaders..." | "Many leaders..." / "Leaders in this context..." |
| "Obviously..." | Remove — if it were obvious, no study would be needed |

### First Person
| Context | Rule |
|---------|------|
| Literature review | Avoid first person; use "the researcher" or passive voice |
| Methodology (most programs) | "the researcher" |
| Role of the researcher / positionality | First person is appropriate and expected |
| Discussion / implications | "the researcher" unless institution permits "I" |

### Passive vs. Active Voice
- **Quantitative/science:** Passive is conventional in methods sections ("Data were collected...")
- **Qualitative/social science:** Active is often preferred ("The researcher conducted...")
- **Lit review:** Either is acceptable; prioritize clarity

### Vague Pronoun References
| Weak | Stronger |
|------|---------|
| "This is important because..." | "This finding is important because..." |
| "It shows that..." | "The data show that..." |
| "They found that..." | "Smith and Jones (2021) found that..." |

## Process

### Step 1 — Receive the Text
The learner pastes a passage (any length) that needs editing.

### Step 2 — Diagnose Problems
Scan for:
- Contractions
- Informal vocabulary
- First-person where it shouldn't be
- Overstatements without hedging
- Vague pronouns
- Weak verbs
- Unsupported claims

### Step 3 — Revise
Return the revised text with tracked changes or side-by-side comparison.

### Step 4 — Annotate Key Changes
For each major edit, briefly explain why:
> "Changed 'a lot of' to 'a substantial proportion of' — more precise register"
> "Added hedge 'may' — the original stated this as certain without evidence"

## Output Format
1. **Original text** (quoted)
2. **Revised text** (clean version)
3. **Change log** (numbered list of significant changes and rationale)
4. **Patterns to watch** (recurring issues the learner should self-monitor)
